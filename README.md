# 🚀 DAES - Diretório Acadêmico de Engenharia de Software
**Universidade de Caxias do Sul (UCS) - Campus Bento Gonçalves**

Bem-vindo ao repositório oficial do site do **1º Diretório Acadêmico de Engenharia de Software** da UCS Bento. Nossa missão é ser a ponte entre os alunos, a universidade e o mercado de tecnologia da Serra Gaúcha.

## 📌 Sobre o Projeto
Este site serve como a central digital oficial para os estudantes do curso. Aqui, os alunos podem encontrar:
- 📚 Links rápidos para nosso Drive de resumos e provas antigas.
- 📅 Mural com os próximos eventos (palestras, hackathons, torneios de ping pong).
- 💬 Acesso direto ao nosso servidor oficial no Discord.
- 📢 Canais de ouvidoria e contato com a diretoria.

## 🛠️ Tecnologias Utilizadas
Como um bom projeto de Eng. de Software, mantivemos a stack simples, rápida e responsiva:
- **HTML5** (Estruturação semântica)
- **Tailwind CSS** (Estilização via CDN para máxima velocidade)
- **FontAwesome** (Ícones)
- **GitHub Pages** (Hospedagem gratuita e CI/CD integrado)
